## Pattern Reminder

- **Prefix Array** → use when you need values from the left side
- **Suffix Array** → use when you need values from the right side
- **Prefix + Suffix** → use when a problem needs both sides of each index

## Common Use Cases
- range sum
- left max / right min
- product except self
- equilibrium index
- subarray counting
